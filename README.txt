INSTUCTION TO RUN THIS PROJECT :   

BEFORE RUN CHANGES OCCURE:
1)    COPY THIS PROJECT TO YOUR SYSTEM AND CLIENTS SYSTEMS.
2)    CHANGE IP ADDRESS IN SERVER.PY FILE . (GIVE THE IP ADDRESS OF SERVER SYSTEM)
3)    CHANGE THE IP ADDRESS IN C.PY FILE . (GIVE THE IP ADDRESS OF CLIENT SYSTEM)
4)    CHANGE THE IP ADDRESS IN CLIENT1.PY FILE . (GIVE THE IP ADDRESS OF SERVER SYSTEM)

NOW RUNNING TIME : 
(NOW CHECK GLOBAL PATH OF PYTHON IS SET BEFORE RUNNING )
1) RUN ALPHA_SERVER (BATCH FILE) EXECUTABLE FILE.
2) RUN APLPHA_SMART_CLIENT EXECUTABLE FILE IN MULTIPLE LAPTOPS .
3) PRESS SEND REQUEST BUTTON ON THE CLIENTS SIDE .
4) WHOLE CLIENTS LIST IN SHOWN IN SERVER SIDE .
5) NOW GIVE THE URL AND DESTINATION PATH TO DOWNLOAD ANY FILE FROM INTERNET.
6) SELECT CLIENT , WHICH YOU WANT TO SEND DATA AND ALSO ACCEPT THE DOWNLOAD REQUEST (ACKNOWLDEGEMENT) ON CLIENT SIDE.
7) NOW PRESS START DOWNLOAD ON SERVER SIDE.
8) FILE DOWNLOAD PROCEED .
9) YOUTUBE DOWNLOADER IS ALSO THE KEY FEATURE OD THIS PROJECT. TYPE URL OD YOUTUBE VIDEO AND OUTPUT IS HIGH QUALITY VIDEO FROM YOUTUBE .
10) FILE SEND BUTTON FOR SEND FILE FROM SERVER TO CLIENT . FIRST PRESS ACCEPT BUTTON FROM CLIENT SIDE THEN FILE SEND BUTTON FROM SERVER SIDE .
11) GAME IS ALSO AVALIABLE FOR ENTERTAINMENT . (FOR BOTH SERVER AND CLIENT )
12) SECURED NETWORK . LOCK SCREEN OPTION AVAILABLE IN BOTH SIDE .
